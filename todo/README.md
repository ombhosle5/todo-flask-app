# 📝 Flask To-Do App with PostgreSQL

This is a simple To-Do list app built using Flask, PostgreSQL, and deployed on Render.

---

## 🚀 Features

- ✅ Add tasks
- 🗑️ Delete tasks
- ✔️ Mark tasks as complete/incomplete
- 👤 User login system

---

## 🧠 Tech Stack

- Backend: Flask
- Database: PostgreSQL
- Hosting: Render
- Language: Python
